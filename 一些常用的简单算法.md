# Some kinds of Simple Algorithms

## 判断素数

1. ### 定义法及其优化

   直接让n除以[2，n-1]中的所有数，如果有余数为0者，则为和数

   ```c++
   for (int i = 2; i < x-1 ; i++)
   	{
   		if (x % i == 0)
   			return 0;
   	}
   		return 1;
   //注意，这个方法对4不适用，应该单独列出if(i==4) return 0;
   ```

   实际上还可以优化，范围是[2，√n]，并且优化效果十分显著

   

2. ###   取模法  

   当n>=6时，n可以表示为6x,6x+1,6x+2,6x+3,6x+4,6x+5中的一种，其中6x,6x+2,6x+3,6x+4肯定不是素数，剩下的再利用定义法即可
   
   ```c++
   //特殊的单列出来
   	if (x == 2 || x == 3 || x == 5) return 1;
   	if (x == 4) return 0;
   //计算是不是6x,6x+2,6x+3,6x+4的形式
   	int _left = x % 6;
   	if (_left == 0 || _left % 6 == 22 || _left == 3 || _left == 4) return 0;
   //剩下的再做判断
   	for (int i = 2; i < x / 2; i++)
   	{
   		if (x % i == 0)
   			return 0;
   	}
   	return 1;
   ```
   
   当数据量很大时，这种方法会优于定义法
